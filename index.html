About me

I am a Computer Engineering graduate from the Georgia Tech, passionate about work that pushes the boundaries of human sensory experiences and bridges the gap between the physical and digital world - making interactions with the cyber world more intuitive. I work with embedded systems, signal processing, and machine learning to engineer such experiences with a strong focus on human-centered design.

Skills

  • Languages - English (native), Hindi/Urdu (native), Arabic (basic), French (basic)
  • Programming - C, C++, C#, Python, Java, MATLAB, Assembly, VHDL, HTML, CSS, JavaScript
  • Hardware - ARM Mbed, Arduino, Raspberry Pi, TI Launchpad, Intel Edison, FPGA
  • Instrumentation - Oscilloscope, Spectrum Analyzer, Logic Analyzer, Function Generator
  • Protocols - SPI, I2C, USB, UART, RS-232, WiFi, Bluetooth, TCP/IP
  • Operating Systems - Windows, Linux, real-time OS (mbed RTOS, TI-RTOS)
  • Software - Quartus, ModelSim, Adobe Photoshop, Adobe Illustrator, AutoDesk Fusion360
  • Rapid Prototyping - soldering, 3D printing, laser cutting, waterjet cutting, woodworking
  • Version Control - Git
